This directory contains the ASCAT Level 2 read software written in the IDL
programming language. 'ascat_l2_main' is the main program which calls the
subroutine 'ascat_l2_read'. 

If you have any further questions or wish to report any issues with using the
data, read software, or documentation, please contact User Services:
podaac@podaac.jpl.nasa.gov
